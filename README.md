# Machine Learning Education

#### MLE-1:
Разработать трансформер обработка данных.
Требования к трансформеру:
- Определение гранулярности данных временного ряда;
- Изменение гранулярности данных временного ряда;
- Обработка пропусков через настройку метода;
- Обработка выбросов через настройку метода;

#### MLE-2:
- Установить дискретизацию тренда по месяцам;
- Выполнить отбор признаков на основе корелляции по их значимости (> 0.3). Выбрать корелляцию по Спирману;
- Учесть тренд как отдельный признак (feature). Можно нормализовать, чтобы перейти к коэффициентам;
- Сгенерировать модели линейной регрессии и XGBoost на основе полученных признков. Соотношение тренировочной и тестовой выборок - 90/10;
- Визулизировать метрику MAE на 10 графиках;

#### MLE-3:
- Рассчитать метрику forecast accuracy. forecast accuracy — весь ряд предсказанный и делим на реальное значение;
- Рассчитать метрику MAPE. Так как MAPE показывает ошибку, то forecast accuracy = 1 - MAPE;
- Выбрать лучшую модель по метрике MAE, но показать одну из метрик выше;
